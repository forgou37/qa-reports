## What I Do

I audit open-source and indie SaaS projects for security vulnerabilities.

Each report includes:
- **Live PoC** — curl-verified against the running app, not just theory
- **CVSS scores** — for every Critical and High finding
- **Root cause** — exact file and line number in the source
- **Remediation code** — working fix, not just a description

Focus areas: auth bypass, broken access control, missing rate limiting, information disclosure, SSRF, injection, insecure configurations.
